Module PDF_Extraction.utils.constants

This module defines constants, enumerations, and utility functions for PDF extraction and processing.

Classes

Constants: Contains various constants used throughout the application, including date patterns, directory paths, and other literals. LatticeType: Enum representing different types of lattice structures in PDFs. MatchType: Enum representing different types of matching strategies. MutliProcessingStrategy: Enum representing different multiprocessing strategies. FileType: Enum representing different file types and their associated extensions and directories.

Constants

DATE_PATTERNS: List of regex patterns for matching various date formats. combined_pattern: Compiled regex pattern combining all date patterns. PDF_DIR: Directory path for PDF files. EXCEL_DIR: Directory path for Excel files. CSV_DIR: Directory path for CSV files. TXT_DIR: Directory path for text files. TXT_OLD_DIR: Directory path for old text files. JSON_DIR: Directory path for JSON files. PYTEST_PDF: Directory path for pytest PDF files. PYTEST_CSV: Directory path for pytest CSV files. PYTEST_EXCEL: Directory path for pytest Excel files. HUNDRED: Constant value 100.0. DATE_COUNT_THRESHOLD: Threshold value for date count. PDF_PAGE_COUNT_THRESHOLD: Threshold value for PDF page count. PRINT_LOG: Flag to enable or disable printing logs. TIME_FUNCTION_ENABLED: Flag to enable or disable timing functions. MEMORY_FUNCTION_ENABLED: Flag to enable or disable memory functions. KEYWORDS_IGNORE_MERGING: List of keywords to ignore during row merging.
